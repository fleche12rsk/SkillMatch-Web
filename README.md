# SkillMatch

Criar um nome para o seu software:
SkillMatch

Descrever qual o problema ele resolve:
Ele faz uma simulação mais próxima do real de vagas fictícias visando ser o máximo real possível com o candidato que esta procurando uma vaga de emprego,
assim facilitando pessoas a souberem suas chances de serem contratadas antes mesmo de sse inscreverem para a vaga de emprego.
